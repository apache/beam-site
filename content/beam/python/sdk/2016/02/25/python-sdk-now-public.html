Dataflow Python SDK is now public!

Feb 25, 2016 â€¢ James Malone [@chimerasaurus]

When the Apache Beam project proposed entry into the Apache Incubator the proposal included the Dataflow Java SDK. In the long term, however, Apache Beam aims to support SDKs implemented in multiple languages, such as Python.

Today, Google submitted the Dataflow Python (2.x) SDK on GitHub. Google is committed to including the in progress python SDK in Apache Beam and, in that spirit, weâ€™ve moved development of the Python SDK to a public repository. While this SDK will not be included with the initial (incubating) releases of Apache Beam, our we plan on incorporating the Python SDK into beam during incubation. We want to take the time to implement changes from the technical vision into the Java SDK before we introduce a Python SDK for Apache Beam. We believe this will allow us to work on the model and SDKs in an ordered fashion.

You can look for the Apache Beam Python SDK in the coming months once we finish forking and refactoring the Java SDK.

apache_beam.typehints.intrinsic_one_ops module¶

Defines the actions intrinsic bytecodes have on the frame.

Each function here corresponds to a bytecode documented in https://docs.python.org/3/library/dis.html . The first argument is a (mutable) FrameState object, the second the integer opcode argument.

Bytecodes with more complicated behavior (e.g. modifying the program counter) are handled inline rather than here.

For internal use only; no backwards-compatibility guarantees.
